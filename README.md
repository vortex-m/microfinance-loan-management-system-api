# Microfinance Loan Management System API

A Java-based backend system designed to manage microfinance loan operations efficiently. This project aims to streamline loan processing, user management, and financial tracking for small-scale lending institutions.

## 🚀 Tech Stack

- Java (Spring Boot)
- Maven
- MySQL (planned)
- RESTful APIs (in progress)

## 📌 Features (Planned)

- User Management (Manager, Loan Officers, Loan Agent, Customers)
- Loan Application Processing
- EMI Calculation & Tracking
- Repayment Management
- Fraud Detection (basic rules / AI-based later)
- Role-Based Access Control
- Notification System (SMS/Email)
